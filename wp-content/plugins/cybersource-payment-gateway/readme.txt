Payment Gateway from CyberSource

== Installation ==
1. Download the WooCommerce CyberSource plugin zip file and extract to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Go to WooCommerce backend settings page
4. Navigate to Payments tab
5. Then go to settings of CyberSource Payment Gateway. Fill the form with credentials and save changes then you are ready.
