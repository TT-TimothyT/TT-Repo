This plugin maps gravity forms fields to NetSuite form fields and pushes data into NetSuite upon for submission.

== Description ==

This plugin allows for multiple forms to be mapped into NetSuite and pushes data into NetSuite by hooking the Gravity
forms "gform_after_submission" hook.

== Installation ==

1. Upload the `trek-form-manager` directory to your plugins directory (by default: `/wp-content/plugins/`)
1. Activate the plugin through the 'Plugins' menu in WordPress
